设置Action权限

  • 回到XXX-Wiki仓库页面

  • 点击右上角的Settings按钮

  • 点击左侧的Actions按钮

  • 点击General按钮

  • 翻到最下方的Workflow permissions部分

  • 勾选Read and write permissions

  • 勾选Allow GitHub Actions to create and approve pull requests

  • 点击Save按钮
